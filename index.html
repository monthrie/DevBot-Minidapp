Welcome to MiniDevBot

MiniDevBot is your AI-powered development assistant on the Minima blockchain.

How it works:

  • This bot is free to use - but each query costs around $0.12 to run - consider donating to keep the service running.
  • You can ask almost anything... at the moment it struggles with the building of entire complex minidapps, but finds simple minidapps pretty easy.
  • You can also ask it to compile your minidapp and download it (this only works when its given you an entire index.html file).

Tips for best results:

  • If you want a plan, ask for a plan, if you want a full minidapp, ask for a full minidapp.
  • Use code snippets when discussing specific code issues.
  • Mark follow-up questions to maintain context in the conversation.

Happy coding with MiniDevBot!
